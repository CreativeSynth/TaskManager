# TaskManager
테스크를 수집하고, 모아놓는 레포

- number_1: 숫자->한국어 변환 능력 평가
- number_2: 한국어->숫자 변환 능력 평가
- ko_quiz: 네이버 단어사전 퀴즈 평가
- summarization: 한 문장으로 요약문 생성 능력 평가
